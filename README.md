## üöÄ Getting Started (If you absolutely must)

If you're bored, please clone this. If you find a bug, please raise an issue. If you find a spelling mistake, please fix it yourself. I'm tired.

### 1\. The Ritual

You need Node.js. If you don't, you're not in the target demographic, and this README is already too advanced for you.

```bash
# Clone this bad boy down
git clone https://github.com/pintoraj/Pinto-Portfolio.git

# Navigate to the epicenter of potential
cd Pinto-Portfolio
```

### 2\. The Incantation

Pick your package manager. They all do the same thing:

```bash
npm install 
# or 
yarn 
# or 
pnpm install
# or 
bun install 
```

### 3\. The Grand Opening

Run the development server and behold the magic:

```bash
npm run dev
```

Open [http://localhost:3000](https://www.google.com/search?q=http://localhost:3000) in your browser. If it doesn't work immediately, that's just a feature, not a bug. It's preparing you for the realities of software development.
